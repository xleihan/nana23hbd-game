> **⚡️ Y2K 粗野主義街頭潮流 —— 白底紅點綴改版上線！**

挑戰極限節奏感！**2K 節奏高手** 是一款專為瀏覽器設計的高流暢度音樂節奏遊戲。本專案精選本地街頭潮流音軌，透過 Web Audio API 於客戶端即時進行音訊波形與能量分析 (Onset Detection)，動態生成獨一無二的專屬節奏譜面！

---

## 🎮 遊戲特色

* **🎹 100% 本地音訊即時分析**：內建歌曲庫，點擊即時以 Web Audio 提取 PCM 能量，完全不依賴後端生成譜面。
* **❤️ 潮酷 Y2K 粗野主義視覺**：全新的 **白底紅點綴 (White & Red Madness)** 風格。高對比粗黑線條、厚重硬陰影、毛玻璃 macOS 框，以及淡紅色的動態透視網格與 CRT 復古掃描線，帶來強烈的潮流衝擊力。
* **🔥 Madness 狂熱模式**：擊中音符積累狂熱能量，蓄滿 100% 觸發狂熱爆發！在狂熱模式中獲得 2X 得分加成與絢麗的霓虹噴灑粒子特效。
* **📱 鍵盤與觸控雙模支援**：支援電腦鍵盤方向鍵與行動端滑動手勢，隨時隨地開啟律動挑戰。

---

## 🕹 遊戲操作指南

### 電腦端 (鍵盤)
當音符落入最下方的判定區時，按下對應的方向鍵：
* `←` 左箭頭 - 第一軌
* `↓` 下箭頭 - 第二軌
* `↑` 上箭頭 - 第三軌
* `→` 右箭頭 - 第四軌

### 行動端 (觸控手勢)
在遊戲畫面上透過滑動進行對應判定：
* `向左滑動` - 第一軌 (←)
* `向下滑動` - 第二軌 (↓)
* `向上滑動` - 第三軌 (↑)
* `向右滑動` - 第四軌 (→)

## ⚙️ 本地運行指南

### 先決條件
* 安裝 [Node.js](https://nodejs.org/) (建議 v18 以上)

### 運行步驟

1. **複製並安裝依賴**：
   ```bash
   npm install
   ```

2. **設置 API 金鑰 (非必須，僅在需要與 AI Studio 連接時)**：
   複製 `.env.example` 為 `.env.local` 並填入 `GEMINI_API_KEY`。

3. **啟動開發伺服器**：
   ```bash
   npm run dev
   ```
   啟動後在瀏覽器開啟 `http://localhost:5173` 即可遊玩！
